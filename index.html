EX1: Web Services

Watch grep Video on Drive

Procedure:

Create Web Service:

1. Open NetBeans IDE 8.1.
2. Go to File → New Project → Choose Java Web > Web Application.
3. Name the project calcws → Select GlassFish v2 and Java EE5 → Click Finish.
4. Right-click calcws → New > Web Service → Name: calwebservice, Package: calculator.
5. Right-click calwebservice → Add Operation:
   o Name: add, Return type: int
   o Add two parameters: a (int), b (int)
   o Change return 0 to return a + b
6. Repeat step 5 for sub, mul, and div operations.
7. Right-click project → Clean and Build → Ensure BUILD SUCCESSFUL.
8. Right-click project → Deploy.
9. In Services tab, go to Web Services → calwebservice → Right-click → Test Web Service.
10. Enter input values and test operations. Click WSDL to view service definition.

Create Web Service Client:

1. Create a new Web Application named calcwsclient.
2. Right-click project → New > Web Service Client.
3. Browse and select the previously created calwebservice project.
4. If error appears, fix it by adding xendorsed="true" at the end of the error line.
5. Right-click Web Pages → New > JSP → Name it client.jsp.
6. In client.jsp, remove <h1> and drag-drop the add operation from Web Service References.
7. Modify the generated code to set sample values like a = 5, b = 13.
8. Clean, Build, and Deploy the project.
9. Right-click client.jsp → Run File → View output.
10. Repeat for sub, mul, and div by creating separate JSP files.

Result:

The calculator web service using the SOAP protocol was successfully implemented and tested.

EX6: Setup Single Node Hadoop Cluster

Procedure (Simplified):

Setup:
1. Check Java: javac -version
2. Install Hadoop 2.8.0 and extract to C:\Hadoop-2.8.0
3. Set Environment Variables:
   o JAVA_HOME → Java bin path
   o HADOOP_HOME → Hadoop bin path
   o Add both to system PATH

Configuration:
1. Edit core-site.xml:
<property>
  <name>fs.defaultFS</name>
  <value>hdfs://localhost:9000</value>
</property>

2. Rename mapred-site.xml.template → mapred-site.xml and add:
<property>
  <name>mapreduce.framework.name</name>
  <value>yarn</value>
</property>

3. Create folders:
C:\Hadoop-2.8.0\data
C:\Hadoop-2.8.0\data\datanode
C:\Hadoop-2.8.0\data\namenode

4. Edit hdfs-site.xml:
<property>
  <name>dfs.replication</name>
  <value>1</value>
</property>
<property>
  <name>dfs.namenode.name.dir</name>
  <value>/hadoop-2.8.0/data/namenode</value>
</property>
<property>
  <name>dfs.datanode.data.dir</name>
  <value>/hadoop-2.8.0/data/datanode</value>
</property>

5. Edit yarn-site.xml:
<property>
  <name>yarn.nodemanager.aux-services</name>
  <value>mapreduce_shuffle</value>
</property>
<property>
  <name>yarn.nodemanager.auxservices.mapreduce.shuffle.class</name>
  <value>org.apache.hadoop.mapred.ShuffleHandler</value>
</property>

EX7: Install Hadoop on Ubuntu

Procedure (Simplified):

1. Update system and install SSH:
sudo apt-get update
sudo apt-get install ssh

2. Install Java:
sudo apt install openjdk-8-jdk

3. Create a new user and set password:
sudo adduser hadoop
su - hadoop

4. Generate SSH keys and add to authorized_keys:
ssh-keygen -t rsa
cat ~/.ssh/id_rsa.pub >> ~/.ssh/authorized_keys

5. Test SSH:
ssh localhost

6. Download and extract Hadoop:
wget https://archive.apache.org/dist/hadoop/core/hadoop-3.1.2/hadoop-3.1.2.tar.gz
tar -xvzf hadoop-3.1.2.tar.gz
mv hadoop-3.1.2 hadoop

7. Set environment variables in ~/.bashrc:
export HADOOP_HOME=/home/hadoop/hadoop
export JAVA_HOME=/usr/lib/jvm/java-8-openjdk-amd64
export PATH=$PATH:$HADOOP_HOME/bin:$HADOOP_HOME/sbin

8. Apply the changes:
source ~/.bashrc

EX8: MapReduce Word Count Program

Algorithm (Shortened):

1. Mapper:
   o For each line in input, split into words.
   o Emit (word, 1) for each word.
2. Reducer:
   o For each word, sum all values.
   o Output: (word, total count)

Procedure (Simplified):

1. Write Mapper class to emit each word with count 1.
2. Write Reducer class to sum counts for each word.
3. Compile and create JAR file.
4. Run Hadoop command to execute the job:
hadoop jar WordCount.jar WordCount input output
5. View output:
hdfs dfs -cat output/part-r-00000
